# 🤖 Machine Learning Projects Repository

Welcome to my Machine Learning Projects Repository 🚀

This repository contains multiple end-to-end Machine Learning projects built using Python, Scikit-learn, Pandas, NumPy, and various Data Analysis & Visualization libraries.

The projects focus on solving real-world problems using:
- 📊 Data Analysis
- 🧹 Data Preprocessing
- 🤖 Machine Learning Models
- 📈 Model Evaluation
- 🚀 Deployment using Streamlit & Hugging Face

---

# 📂 Projects Included

## 🌫️ Air Quality Prediction
Predicts air quality levels using environmental and pollution-related parameters.

### 🔧 Concepts Used
- Regression Models
- Data Cleaning
- Feature Scaling
- EDA & Visualization

---

## ⚡ EV Charging Risk Prediction
Predicts whether an EV user belongs to:
- 🔴 High Default Risk
- 🟢 Low Default Risk

using charging behavior, financial indicators, and user activity patterns.

### 🔧 Concepts Used
- Logistic Regression
- Decision Tree
- Feature Encoding
- Pipeline Implementation
- Classification Models

---

## 🎬 Netflix Content Classification
Analyzes and classifies Netflix content using metadata and content-related features.

### 🔧 Concepts Used
- Data Analysis
- Classification
- Visualization
- Feature Engineering

---

## 🎓 Student Course Completion Prediction
Predicts whether a student is likely to complete a course successfully based on academic and behavioral attributes.

### 🔧 Concepts Used
- Classification Models
- Data Preprocessing
- EDA
- Model Evaluation
