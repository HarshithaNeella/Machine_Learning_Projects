# ⚡ EV Charging Risk Prediction using Machine Learning

## 📌 Overview

Built a Machine Learning classification system to predict whether an EV user has:

* 🔴 High Default Risk
* 🟢 Low Default Risk

using EV charging behavior, financial indicators, and user activity patterns.

---

## 🎯 Problem Statement

Traditional financial risk systems often ignore EV-specific user behavior such as:

* Charging frequency
* Energy usage
* Missed payments
* Charging efficiency
* User engagement

This project predicts customer default risk using Machine Learning to help improve:

* 💳 Loan approval systems
* 📊 Financial risk analysis
* ⚡ EV customer analytics

---

## 📂 Dataset Features

### 👤 Customer Details

* Age
* Income Level
* City Tier

### 🚗 EV Information

* EV Type
* Battery Capacity
* Distance Travelled

### ⚡ Charging Behavior

* Charging Sessions
* Charging Time
* Charging Efficiency
* Charging Location

### 💰 Financial Features

* Loan Taken
* Missed Payments
* Average Charging Cost

### 🎯 Target Variable

`High_Default_Risk`

* 0 → Low Risk
* 1 → High Risk
